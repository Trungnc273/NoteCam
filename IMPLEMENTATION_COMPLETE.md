## 🔧 Technical Details

### Storage Strategy
- **Hive**: Notes storage (fast, type-safe, supports encryption)
- **SharedPreferences**: Simple settings (haptic, display mode, etc.)
- **Flutter Secure Storage**: PIN storage (encrypted)

### Note Model
```dart
Note {
  String id;              // Unique ID (timestamp)
  String title;           // Note title
  String body;            // Note content
  String tag;             // Category tag
  bool starred;           // Star status
  DateTime createdAt;     // Creation timestamp
  DateTime updatedAt;     // Last update timestamp
}
```

### Auto-Save Behavior
- Notes auto-save khi user tap back button
- Empty notes (no title and no body) không được save
- Notes với empty title được save với title "Không có tiêu đề"

### Search Algorithm
- Case-insensitive search
- Searches both title and body
- Returns notes sorted by updatedAt (newest first)
